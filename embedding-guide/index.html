Embedding orb in your site is fairly simple.

First, add the orb.autoregister meta (This tells Orb to auto-register the orb-edit element):

<meta name="orb.autoregister" value="true">

Then, import orb:

<script src="/path/to/orb.js"></script>

Next, add an orb-edit tag to your HTML:

<orb-edit name="myEdit"><!-- Your initial Orb code here --></orb-edit>

And a run button:

<button id="runBtn">Run</button>

In the JS:

var myEdit = new orb.editor.Editor('myEdit')
  , runBtn = document.getElementById('runBtn');

runBtn.addEventListener('click', function() {
  myEdit.run();
});

You can also make a custom logger:

var myLogger = {
  log: function(message) {
    document.getElementById('log').innerHTML += message;
  }
};

And pass it in to the orb.editor.Editor constructor:

...
var myEdit = new orb.editor.Editor('myEdit', myLogger)
...
